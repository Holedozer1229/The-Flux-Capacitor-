## Overview

Welcome to The Flux Capacitor, a groundbreaking project that transforms a vintage 8-track magnetic tape player into a quantum-inspired audio effects generator. Inspired by the *Back to the Future* flux capacitor’s legendary 1.21 gigawatts, this system blends retro technology with modern computational physics. Enhanced with Closed Timelike Curve (CTC) simulations, it aims to experimentally prove **vector lattice entanglement** and **quantum electromagnetic field J^4 coupling**, delivering warped audio with harmonic richness and hints of spacetime-like influences.

⚛️ **Core Features:**
- Simulates quantum fields, spacetime curvature, and gravitational waves.
- Integrates CTC feedback for retrocausal temporal effects.
- Modulates 8-track playback via hardware control with real-time sensor feedback.

☢️ **Experimental Goals:**
- Demonstrate **vector lattice entanglement** through a quantum spin network.
- Validate **J^4 coupling** with nonlinear harmonic generation in audio output.
- Explore **CTC influences** manifesting as temporal delays in the signal.

## Files

- **`flux_capacitor.py`**: The main Python script, integrating the `UnifiedSpacetimeSimulator` with CTC enhancements for simulation and hardware control.
- **`arduino/flux_capacitor/flux_capacitor.ino`**: Arduino code managing the 8-track player’s stepper motor, electromagnet, and Hall sensor.
- **`MANUSCRIPT.md`**: Detailed documentation of the theoretical framework, implementation, and experimental design.
